# Zhang Hua's AI Content Factory - OpenClaw Configuration

project: "content-creator-factory"
owner: "zhanghua"

agents:
  - name: "research-agent"
    role: "24/7 Trending Research"
    skills:
      - web_scraping
      - api_integration
    heartbeat: "*/30 * * * *"  # Scans every 30 minutes
    sources:
      - weibo_trending
      - xiaohongshu_explore
      - twitter_trends
      - linkedin_pulse
    output: "daily_hot_topics_report"

  - name: "content-creation-agent"
    role: "Multi-Platform Content Generation"
    skills:
      - natural_language_generation
      - seo_optimization
    trigger: "on_topic_approved"  # Triggers after Zhang Hua confirms topic
    platforms:
      - target: xiaohongshu
        style: "lifestyle/guide"
      - target: wechat_mp
        style: "in-depth long-form"
      - target: twitter
        style: "thread/opinion"
      - target: linkedin
        style: "professional insights"

  - name: "publishing-agent"
    role: "Smart Multi-Platform Publishing"
    skills:
      - social_media_api
      - scheduler
    schedule:
      xiaohongshu: "09:00, 12:30, 20:00"
      wechat_mp: "08:00"
      twitter: "10:00, 15:00, 21:00"  # UTC+8
      linkedin: "09:30"

  - name: "analytics-agent"
    role: "Data Analysis & Strategy Optimization"
    skills:
      - data_scraping
      - nlg_report
    heartbeat: "0 22 * * *"  # Generates daily report every night at 10 PM
    output: "daily_performance_report"

workflow:
  morning_routine:
    - research-agent → daily_report
    - Zhang Hua → 30-min review + topic confirmation
    - content-creation-agent → multi-platform drafts
    - publishing-agent → scheduled publishing
  evening_routine:
    - analytics-agent → daily data report + optimization recommendations

Chapter 06

Reusable Framework

Zhang Hua's practical insights on AI-powered content creation, helping more independent creators replicate her success

  • 1

    Start Small, Iterate Gradually

    Start from the most time-consuming and repetitive parts, gradually introducing agents. Don't pursue full automation from the start — get one agent running first, verify the results, then expand.

  • 2

    Define Agent Responsibilities Clearly, Avoid Chaos

    Set clear, single-purpose responsibilities for each OpenClaw agent. Research stays research, creation stays creation, publishing stays publishing — just like managing a real team.

  • 3

    Leverage Localized Integration Tools

    For users in China, make full use of localized versions like Molili to ensure deep integration and stable operation with platforms like WeChat Official Accounts and Xiaohongshu.

  • 4

    Embrace "Heartbeat" and Proactive Automation

    Use the Scheduler to have agents proactively monitor information sources and trigger tasks, achieving truly unmanned 24/7 operations. This is the critical step from "AI-assisted" to "AI-driven."

  • 5

    Continuous Feedback, Human-AI Collaborative Optimization

    Regularly review agent outputs and provide feedback and fine-tuning instructions. AI doesn't replace you — it amplifies your judgment. Your 30 minutes of daily input determines the quality of the entire system's output.
